## TensorFlow2 计算图 Graph
```
它是包含一系列tensorflow操作的数据结构
	包含tensor张量，代表操作之间流动的数据单位
静态图
装饰器  tf.function()
	这样才可以生成计算图
通过计算图优化过程

作用：
	加速
		因为减少了通信开销并加快了速度

eager模式：
	更容易debug
	恢复方法：model.compile(run_eagerly=True)
	
Polymorphism
	函数的数据类型和形状可以不同
```

## TensorFlow2 训练流程
```
1获取训练数据（很贵，一般是大公司进行训练）
2定义模型
3定义损失函数
4运行训练数据，从目标计算损失
5计算损失的梯度，并使用优化器来调整变量以适应数据
6结果评估
```
