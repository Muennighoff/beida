## 语音识别原理介绍
```
原理
评判指标
经典方法
连接主义的时间分类CTC原理
端到端DeepSpeech2

很多经典方法都被深度学习模型替换了
大作业的数据集来自24条智能电器的语音指令

语音识别ASR面临的挑战：
	口音、语气、腔调、感情
	说话的方式
	词汇表的大小、同音词
	噪声、录音质量
	鸡尾酒会问题（多个人同时说话的识别）
	
波形图
频率图
时频谱图 （一般使用这个）

时频谱图
	优点：
		保留时间和频域的信息
		
评判指标
	词错误率(word error rate) WER
	方法：答案和识别结果对齐
	错误率等于 替换、插入、删除总数 / 标准答案长度
	错误率可能会大于100%
	
5个经典的语音识别方法
	n-gram
	...
	
2006年以前的词错误率基本在10%左右，无法作为工具使用

深度语音识别开始引入，引入之后错误率下降了30%+

语音识别的深度学习：
	输入波形，输出文字
	使用纯粹的深度学习模型
	用足够多的语料与计算机资源来完成大语音识别系统
	
技术路线创新：循环网络处理语音
CTC connectionist temporal classification可以看作是深度学习网络
CTC可以自动解决对齐的过程（之前都是人工手动去标识）
	大幅减少耗时
```

## CTC 介绍
```
解决的问题：对齐
CTC给出所有可能的文字分布[wwo-rrr-lld!](- 是占位符\空白)，然后把所有的重复组合在一起，之后给出最有可能的结果[world!]

tensorflow2 存在 CTC loss, CTC beam search
```
