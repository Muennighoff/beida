##### 取交集的语句

```sql
SELECT DISTINCT R.A
FROM R,S,T
WHERE R.A = S.A OR R.A = T.A
```

以上语句用python写出等价于

```python
for r in R:
	for s in S:  ##len(S)==0,直接跳出循环
        for t in T:
            if r['A'] == s['A'] or r['A'] == t['A']:
                output.add(r['A'])
```

- 并集 UNION (没有重复值)

##### 嵌套查询

Nested queries 嵌套查询

- IN 可以配合 SELECT DISTINCT使用
- ALL
- EXISTS

##### 聚合和分组

- GROUP BY

group by的执行顺序是先用from where确定范围，再根据某一列合并，最后select出想要的列

having语句对于分组后的表进行条件筛选

```sql
SELECT product, Sum(price*quantity) AS TotalSales
FROM Purchase
WHERE date > ‘10/1/2005’
GROUP BY product
HAVING SUM(quantity) > 100
```

### python sqlite3库
##### 创建数据库

```python
import sqlite3
#创建数据库
with sqlite3.connect("university.db") as db: 
    cursor = db.cursor()
```

##### 创建表

```python
#创建表，必须指定数据类型和哪些子域必须为NOT NULL
cursor.execute(""" CREATE TABLE IF NOT EXISTS students( 
id integer PRIMARY KEY, 
name text NOT NULL,
gender text NOT NULL,
age integer NOT NULL,
major text NOT NULL);""")
```

##### 插入数据

```python
#插入固定数据
cursor.execute(""" INSERT INTO students(id, name, gender, age, major) 
VALUES ("001", "A", "MALE", "19", "X")""")
db.commit() #提交到数据库
#一定要commit，不然不能保存
```

##### 读取并展示

```python
#读取数据
cursor.execute("SELECT * FROM students") 
#展示数据
print(cursor.fetchall()) #直接print
for x in cursor.fetchall(): #一个个print
    print(x)
db.close()
```
