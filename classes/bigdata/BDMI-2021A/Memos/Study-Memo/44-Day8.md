### Logit Function

def logit(l):
    out=[]
    for i in l:
     out.append(math.log(i/(1-i)))

#### 损失函数
>损失函数：输出结果与标签值的差异

损失函数的度量函数：
* MSE（均方误差）：常用于回归任务
```python
def MSE(input,lable):
 output=0
 N=len(input)
 for i in input:
  output+=(i-lable)**2
 output/=N
 return output
```
* CE（交叉熵）：常用于分类任务
```python
def CE(test,lable):
 result=0
 N=len(test)
 i=0
 while i<N:
  result+=lable[i]*math.log(test[i])
  i+=1
 return -result
```
