## 快速排序算法

> QuickSort(A):
   If len(A) <= 1:
   return
   Pick some x = A[i] at random. Call this the pivot. 
   PARTITION the rest of A into:
   L (less than x) and
   R (greater than x)
   Replace A with [L, x, R] (that is, rearrange A in this order)
   QuickSort(L)
   QuickSort(R)
>$$
>时间复杂度：O(nlogn)\\
>最差时间复杂度：O(n^2)
>$$

## 二叉搜索树
### 链表（linked list）
>链表结构：数据和指向下一个数据的指针

>Node定义：
```python
class Node:
   def __init__(self,x,next=None):
      self.val=x
      self.next=next
```
### 二叉搜索树（binary search trees）
>二叉搜索树包含一个根节点，每个节点包含两个子节点，右子节点都大于父节点，左子节点都小于父节点
>插入、搜索、删除操作：
>>插入操作&搜索操作：通过遍历找到该数据或者插入位点，然后进行插入操作
>>删除操作：
>>
>>> * 删除节点无子节点，直接删除；
>>> * 有一个子节点，子节点换到父节点位置;
>>> * 有两个子节点，需要把左子节点的最右子节点代替，或者用右子节点的最左子节点代替

## 红黑树
### 红黑树的特征：
* 每个节点都是黑色或者红色
* 根节点是黑色的
* 每个叶子节（NIL）点是黑色的
* 红色节点的子节点是黑色的
* 对于所有节点，从该节点到NIL节点的所有路径有着相同数目的黑节点
>旋转：左旋和右旋操作
>自平衡
