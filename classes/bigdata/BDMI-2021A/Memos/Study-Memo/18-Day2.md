> - 读取函数
> - 写入函数
> - 文件删除函数

## 模块和包

这部分学习了python的包和模块，利用包和模块的封装，是程序变得可复用，、结构清晰、可维护性高。

> - 举例
>
> > ```python
> > import module1
> > from modname import name1
> > ```
>
> - 调用方法
>
> > ```python
> > from food_store.product.fruit.banana import Banana
> > from food_store.product.fruit import banana
> > ```
