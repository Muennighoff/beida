## 第一次课堂小结

> 本次课主要讲解 Python 相关语法知识，这里做简要总结。

### Python 数据类型

包括列表、集合、字典、元组

### 控制语句

#### 条件语句：if

配合 elif 和 else 进行条件控制

#### 循环语句：for

后面跟着需要循环的列表，也可以使用 range 生成列表

#### 循环语句：while

检验条件是否为真，若为真则进行循环

#### 函数：def

使用关键字 def 定义一个函数，并进行调用：

```python
if __name__ == '__main__' :
    myFunc()
```

可以在 `__main__` 函数中定义一个主函数

#### 类：class

与其他语言中的类相似，只不过：

- 定义构造函数的方式是定义 `__init__` 方法
- 定义方法时，第一个参数永远是 `self`，代表对象自身
