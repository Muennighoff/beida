#### 条件语句 if, elif, else

```python
if x>100:
	print()
elif x==100:
	print()
else:
	print()
```

#### for 循环

```python
for i in list:
	print(i)
for i in range(n):
	print(i)
```

#### while 循环

```python
while i<500:
	print(i)
	i+=1
```

#### 函数 Functions

- 函数是将语句集合在一起的基本组件

```python
#函数定义：
def function(number):
	print('This is' +number)
    
#函数调用：
if __name__ == '__main__' :
	function(5)
```

#### 匿名函数 lambda

- 只有一行的函数

```python
add = lambda x, y: x + y

print(add(3, 5))
# Output: 8
```
