## 3.插入排序 Insertion sort

 		给定一个序列，从第二个元素开始，与之前的元素进行比较，若小于该元素，则继续和之前的元素比较，直到找到比自己小且最大的元素，就可以将被比较的元素插入。该方法的复杂度为o（n^2）

> ```
> def InsertionSort(A):
>     for i in range(1, len(A)):
>         current = A[i]
>         j = i-1
>         while j >= 0 and A[j] > current:
>             A[j+1] = A[j]
>             j -= 1
>             A[j+1] = current
> ```

## 4.合并排序 merge sort

​		最大复杂度0(n log n)

> ```
> import random
> A = [2, 5, 9, 12, 16, 18]
> B = [1, 3, 7, 11, 13, 15, 17, 19, 20, 22]
> 
> def merge(A, B):
>     C = []
>     i = 0
>     j = 0
>     while i < len(A) and j < len(B):
>         if A[i] < B[j]:
>             C.append(A[i])
>             i += 1
>         else:
>             C.append(B[j])
>             j += 1
>     while i < len(A):
>         C.append(A[i])
>         i += 1
>     while j < len(B):
>         C.append(B[j])
>         j += 1
>     return C
> 
> def mergesort(A):
>     n = len(A)
>     if n <= 1:
>         return A
>     L = mergesort(A[0:n/2])
>     R = mergesort(A[n/2:n])
>     return merge(L,R)
> 
> C = merge(A, B)
> random.shuffle(C)
> C = mergesort(C)
> print(C)
> ```
